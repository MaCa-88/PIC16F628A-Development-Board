# PIC16F628A-Development-Board
Diseño y fabricación de tarjeta de desarrollo para microcontrolador PIC16F628A. El proyecto integra el flujo completo de diseño electrónico en Proteus y la preparación de archivos G-Code mediante FlatCAM para fabricación por fresado CNC de alta precisión. Optimizado para prototipado rápido sin procesos químicos.
